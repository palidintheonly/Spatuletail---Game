PRIVACY POLICY

Spatuletail - Game

Effective Date: January 19, 2025

Our Commitment to Transparency: We collect ONLY the data explicitly listed in this policy. We do not collect, track, or store any additional personal information beyond what is necessary for the game to function. We are committed to being truthful and transparent about all data practices.

1. Introduction

This Privacy Policy explains how Spatuletail - Game ("we," "us," or "the Game") collects, uses, and protects your information when you use our online battleship game.

By using the Game, you agree to the collection and use of information in accordance with this policy.

2. Information We Collect

We are committed to collecting ONLY the minimum data necessary for the Game to function. Below is a complete and exhaustive list of ALL data we collect:

2.1 Data We Collect

Data Type Purpose Storage Location
Player Names Display in-game and on leaderboards Server (JSON files)
Game Statistics Track wins, losses, games played Server (JSON files)
Leaderboard Data Display rankings and achievements Server (JSON files)
IP Addresses Rate limiting and abuse prevention Server memory (temporary)
Session Data Maintain game state during active sessions Server memory (temporary)
Game Logs Track game events and improve gameplay Server (JSON files)

2.2 Data We DO NOT Collect

We explicitly DO NOT collect:

  • Email addresses or contact information
  • Passwords or authentication credentials
  • Payment information or financial data
  • Device identifiers or fingerprints
  • Location data or geolocation
  • Cookies or tracking technologies (beyond session management)
  • Social media profiles or third-party account data
  • Browsing history or external website activity
  • Any other personal identifiable information not explicitly listed above

3. How We Use Your Information

We use the collected data exclusively for:

  • Game Functionality: Enabling multiplayer features, matchmaking, and game state management
  • Leaderboards: Displaying player rankings and statistics
  • Security: Preventing abuse, cheating, and server overload through rate limiting
  • Improvement: Analyzing game logs to improve gameplay and fix bugs
  • Administration: Managing active games and monitoring server health

4. Data Storage and Retention

4.1 Storage Methods

  • Persistent Storage: Leaderboards and game logs are stored in JSON files on the server
  • Temporary Storage: Session data and IP addresses are stored in server memory only during active sessions
  • Automatic Cleanup: Game logs are limited to the most recent 2,000 entries, and event logs to 2,000 entries

4.2 Data Retention Periods

Data Type Retention Period
Session Data Duration of active session only
IP Addresses Duration of active session only
Leaderboard Data Indefinitely (top 100 entries per mode)
Game Logs Rolling window (last 2,000 games)
Event Logs Rolling window (last 2,000 events)

5. Data Sharing and Disclosure

We do NOT sell, trade, or share your data with third parties.

The only data that is publicly visible:

  • Player names on leaderboards
  • Win/loss statistics on leaderboards
  • Live game statistics (active players, games played)

6. Security Measures

We implement security measures to protect your data:

  • Rate Limiting: Prevents abuse and DDoS attacks (configurable, default: 100 requests per 15 minutes)
  • Admin Protection: Password-protected admin panel with session timeouts
  • Input Validation: All user inputs are validated to prevent injection attacks
  • Connection Limits: Maximum connections per IP address (default: 10)

7. Your Rights

You have the right to:

  • Access: View your leaderboard data by accessing the public leaderboards
  • Delete: Request removal from leaderboards by contacting us
  • Object: Choose not to participate by not using the Game
  • Transparency: Request information about data we hold (limited to what's listed in Section 2)

8. Children's Privacy

The Game does not knowingly collect personal information from children under 13. If you are a parent or guardian and believe your child has provided us with personal information, please contact us.

9. Data Controller

10. Third-Party Services

Important: All game functionality runs on our server. We do not use external analytics, tracking, or advertising services.
